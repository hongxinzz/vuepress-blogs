    # Promise学习总结

    # api总结

    • Promise 分别成功、失败、等待有三种状态。
        pending: 初始状态，既不是成功，也不是失败状态。
    fulfilled: 意味着操作成功完成。
    rejected: 意味着操作失败。
    • 当返回成功、失败状态时，此前的状态不会更改。
       let p =  new Promise((resolve,reject)=>{
        resolve('success');
        reject(new Error('fail'))
    })
    p.then(res=>{
        console.log(res) //success
    },err=>{
        console.log(err)
    })
    • Promise.all 所有的方法如果成功则回调resolve，如果其中一个错误则返回第一个错误的原因。(如果方法有依赖关系不建议并发执行)
        const promise1 = Promise.resolve(3);
    const promise2 = new Promise((resolve, reject) => {
        setTimeout(resolve, 1000, 'foo');
    });

    const promise3 = new Promise((resolve,reject) => {
        setTimeout(reject,500,new Error('fail'));
    })


    Promise.all([promise1,promise2]).then(res=>{
        console.log(res); // [3,'foo']
    })

    Promise.all([promise1,promise2,promise3])
    .then(res=>{
        console.log(res);
    },err=>{
        console.log(err); //Error: fail
    })
    • Promise.race可以理解成返回队列中第一个返回的值(不管失败还是成功)。
        const promise1 = Promise.resolve(3);
    const promise2 = new Promise((resolve, reject) => {
        setTimeout(resolve, 1000, 'foo');
    });

    const promise3 = new Promise((resolve,reject) => {
        setTimeout(reject,500,new Error('fail'));
    })


    Promise.race([promise1,promise2]).then(res=>{
        console.log(res); // 3
    })

    Promise.race([promise2,promise3])
    .then(res=>{
        console.log(res);
    },err=>{
        console.log(err); //Error: fail
    })
    • Promise.then 具有穿透性。
        //.then具有穿透性
   Promise.resolve(1).then(res=>{
       console.log(res)
       return 1 + res; //如果再这返回成功值，则在下一次.then中可以接收到
   }).then(res => {
       console.log(res)
   })

    // 没有失败态 catch不会阻止
    Promise.resolve(1).then(res=>{
       console.log(res)
       return 1 + res; //如果再这返回成功值，则在下一次.then中可以接收到
   }).catch(err=>{
       console.log(err) // 跳过了 catch
   }).then(res => {
       console.log(res)
   })

    //catch 是监听整条链子上错误的一个钩子函数
   Promise.resolve(1).then(res=>{
       console.log(res)
       return 1 + res; //如果再这返回成功值，则在下一次.then中可以接收到
   }).catch(err=>{
       console.log(err) // catch 相当于整条链子的错误捕获
   }).then(res => {
       console.log(res)
       return Promise.reject(new Error('fail'));
   })

    # 小实战

    /**
* 读取文件
* @param {*} blob
*/
function blob2Url(blob) {
   return new Promise((resolve, reject) => {
       const reader = new FileReader();
       reader.onload = function () {
           resolve(reader.result);
       };
       reader.onerror = function (e) {
           reject(e);
       };
       // 以DataURL的形式读取文件
       reader.readAsDataURL(blob);
   });
}
